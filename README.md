# 🎬 Netflix Clone

A responsive Netflix landing page clone built using **HTML, CSS, and JavaScript**. This project was created for practice to improve frontend development skills by recreating the look and feel of Netflix's homepage.

## 🚀 Live Demo

🔗 https://singhananya15.github.io/netflix-clone/

## ✨ Features

- 🎥 Netflix-inspired landing page
- 🎨 Responsive UI with modern styling
- 📺 Trending movies section
- 💡 "More Reasons to Join" cards
- ❓ Interactive FAQ accordion using JavaScript
- 🔐 Separate Sign In page
- 🖱️ Hover effects and smooth transitions
- 🌐 Deployed using GitHub Pages

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
